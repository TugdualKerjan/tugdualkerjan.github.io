Exploring Barilla's Pasta Varieties

I want to eat all the pasta shapes proposed by Barilla.

Unfortunately, when contacting their customer service I failed to get a satisfactory answer concerning finding a comprehensive list of all their products.

I resorted to using web scrapping to obtain the full list of Barilla products sold on their website, removing duplicates, and finally sorted through 1300 items using google sheets IMAGE() function, noting if it had a number referenced on it or not. You can find the google sheet here. Alternatively download the raw CSV file here.

After going through the list I transformed it into a JSON file that could be loaded and displayed thanks to Javascript and OpenLayers. If you want more details feel free to check out the code here.
