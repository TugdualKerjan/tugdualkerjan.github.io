# [HEATSHARE](https://heatshare.tugdual.fr)
![](calheatmap.png)

I've been using Anki on and off for around 5 years now - one of the things that has allowed me to stay consistent is sharing my consistency with fellow users and friends / family I had convinced to try use the tool ! When looking for a way to share my progress I noticed there existing tools didn't allow for sharing a review heatmap - this changes today with <a href="heatshare.tugdual.fr">heatshare</a>, an Anki plugin that allows anyone to sync their progress and then embed the heatmap into whatever personal blog they have ! My site now integrates my review map so anyone can see where I'm at concerning card reviews across the year 🔥
