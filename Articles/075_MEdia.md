# [MEdia](https://tugdual.fr/MEdia)
![](booksmovies.png)

I've always liked keeping track of and sharing the books, movies and other content that make me who I am. I've yet to find a convenient website that allows me to simply view a file in which I store the relevant media I've consumed without feeling like I wasn't in control of my data. I'm still exploring and there are other features I would like to add, but <a href="https://tugdual.fr/MEdia/">MEdia</a> shows a simple draft of what could be a nice way of sharing MEdia and everything meta around them with friends and family: Reviews, highlighted passages, notes... Initially I turned to <a href="https://dokie.li/">dokie.li</a> for this but found their tool had a really high barrier to entry, lacking the basic ability to open a local file anonymously for a quick edit, something which is now natively available on browsers such as chrome thanks to <a href="https://wicg.github.io/file-system-access/">The file system access API</a>
