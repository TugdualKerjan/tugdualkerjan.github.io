# [NETWORKS IN JAVA](https://github.com/TugdualKerjan/NeuralNetworksInJava)
![](images/typingfinger.png)

My fascination with neural networks started with <a href="https://www.youtube.com/watch?v=qv6UVOQ0F44">this youtube video</a> which still makes me marvel in awe today. It motivated me to take the little knowledge I had of programming games for Minecraft in Java to try and build my own neural networks. Following <a href="https://www.youtube.com/watch?v=d3OtgsGcMLw&list=PLgomWLYGNl1dL1Qsmgumhcg4HOcWZMd3k">this suite of tutorials</a> on youtube, I slowly built my way up to working neural networks ! It remains by far one of my proudest achievements, having persevered through the terrific bad code I wrote and buggy scripts to arrive to something that could read handwriting and output the corresponding number.
