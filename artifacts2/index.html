Artifacts II

I currently use Goodreads and Letterboxd for which books and movies I watch, but don't like having my data stuck in their hands.

This HTML file is all there is to the solution: It can load a URL or local JSON representing the list of media. Anyone can take this HTML file and open it anywhere, it'll just work ™.

This is a small step to having the user own the files instead of letting companies manage them.

Upcoming features: Addition of other video formats, and music. Migrate to the schema.org formats so as to facilitate things by respecting standards.

You can create your own list of media you've consumed and share it however you want: Using a USB, hosting it online, or keeping it in your homeserver. All you need is the html page you're currently looking at.
